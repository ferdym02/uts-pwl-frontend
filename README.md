# UTS PWL (Pemrograman Web Lanjut)

## Cara Menjalankan

### Masuk ke Direktori

```
cd .\uts-pwl-frontend\
```

### Install Depedencies

```
npm i
```

```
npm i react-router-dom axios bulma
```

### Run Program

```
npm start
```

Pastikan sudah terlebih dahulu menjalankan backend
